Trouver de la documentation

  • man commande
  • info commande

(en remplaçant commande par le nom de l'application sur laquelle vous cherchez des informations) 

$ whatis bash
bash (1)             - Interpréteur de commandes GNU Bourne-Again SHell

Gérer les fichiers et les répertoires

Se déplacer dans les répertoires

Se situer : pwd

$ pwd

Aller dans un dossier : cd

$ cd nouveau_repertoire
$ cd www/logs

Les chemins relatifs

$ cd ..
$ cd ../documents

Les chemins absolus

$ cd /home/batman/documents
$ cd /usr/share/doc

Lister les fichiers d'un répertoire

La commande ls

$ ls

$ ls -la
$ ls -ld SomeDir
$ ls -R SomeDir

Créer un fichier ou un répertoire

Créer un fichier vide : touch

$ touch EmptyFile

Créer un répertoire : mkdir

$ mkdir nouveauRepertoire
$ mkdir -p nouveauRepertoire/etSousRepertoire

Supprimer un fichier ou un répertoire

$ rm fichierVide

$ rm -r nouveauDossier

Attention : pas de confirmation sans le paramètre -i

$ rm -ir nouveauDossier

Copier un fichier ou un répertoire

$ cp fichierQuelconque nouveauFichier
$ cp Image.jpg ~/www/images/

Déplacer un fichier ou un répertoire

$ mv fichierQuelconque fichierDestination
$ mv fichierQuelconque ~/documents/
$ mv repertoireQuelconque ~/documents/

Permissions

$ ls -l *.txt
-rw-------  1 batman  batteam  0 Mar 28 12:45 fichierPrivé.txt
-rw-rw-r--  1 batgirl batteam  0 Mar 28 12:47 fichierPublic.txt

Changer les permissions : chmod

$ chmod g+r fichierPrivé.txt

Changer le propriétaire : chown

$ chown batman:batteam privateBatcaveFile.txt

Changer le groupe : chgroup

$ chgrp batteam PrivateBatCaveFile.txt

Quelques astuces

Attention aux espaces !

$ cd "nouveau repertoire avec des espaces"
$ cd nouveau\ repertoire\ avec\ des\ espaces

Revenir dans $HOME

$ cd

Fichiers commençant par un tiret (-)

$ rm \-StupidFile

Contenu des fichiers

Afficher le contenu d'un fichier

En entier

$ cat ~/ToDo.txt

... seulement son début

$ head ~/www/logs/access
$ head -n 20 -f ~/www/logs/error

... ou seulement sa fin

$ tail ~/www/logs/access
$ tail -n 20 ~/www/logs/access
$ tail -f ~/www/logs/error

Rechercher des fichiers

La commande grep

$ grep 'Catwoman' ~/realVillains.txt
$ grep -r 'Catwoman' ~/villains/
$ egrep '(Catwoman|Poison Ivy)' ~/recentCrimes.txt

La commande find


$ find ~/documents/ -name \*.html
$ find ~/www/ -name \*.html -or -name \*.txt -exec grep -H {} \;

Liens

Liens symboliques : ln

$ ln -s monFichier lienVersMonFichier
$ ln -s monRepertoire lienVersMonRepertoire

Liens durs : ln (encore)

$ ln MyFile LinkToMyFile
